# Proyecto HTML: Página con Menú y Tabla

## Descripción
Este proyecto académico consiste en una página HTML estática que sirve como práctica de trabajo por ramas en Git.  
Se han implementado dos mejoras principales: un menú de navegación con enlaces internos y una tabla informativa semántica.  
El flujo de trabajo se ha documentado mediante commits descriptivos y Pull Requests en GitHub.
